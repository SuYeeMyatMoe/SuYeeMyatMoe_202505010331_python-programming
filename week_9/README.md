# IT Helpdesk Ticket Registration System

## 1. Purpose of the Application
The IT Helpdesk Ticket Registration System is a modular Python application designed for the IT Support team at City University Malaysia. It allows students to report technical issues, captures ticket details, dynamically assigns technicians based on priority levels, and displays formatted ticket status information.

## 2. Tech Stack
* **Language:** Python 3

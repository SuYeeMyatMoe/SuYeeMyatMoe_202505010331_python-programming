# Food Delivery System

### 4.1 Purpose of the Application
A modularized backend receipt generator for a food delivery system that calculates totals including service charges and delivery fees.

### 4.2 Tech Stack
* Language: Python 3
* Architecture: Modular programming (separation of concerns)

### 4.3 How to Use
1. Ensure you have Python installed.
2. Navigate to the `week_8` directory.
3. Run the application using: `python main.py`
4. Follow the prompts to enter customer and order details.
